## 확인 못 해 뺀 것 / 조문 번호 없이 쓴 것

- 시행규칙 제44조 각 호(제1호·제1호의2·제2호·제3호·제4호·제6호 등)의 **정확한 호 번호와 세부 문구**는 law-cache에 목차만 있고 조문 본문이 없어(get_law_text API가 조회 시점 NOT_FOUND 반환), 본문에서는 "제44조" 조 단위로만 인용하고 각 호 세부는 요지로만 서술 + fieldVerification 등재.
- 입찰무효 판정에 대한 입찰자의 **명시적 이의·구제 절차**는 국가계약법령에 별도 규정을 확인하지 못해, 무효 이유 확인(규칙 제45조) 수준으로만 연결하고 이의 절차 자체는 fieldVerification로 돌림.
- 전자입찰특별유의서 별표2 자동차단 근거로 표에 표시된 "시행규칙 제40조제4항" 등은 표 원문에 기재된 값으로 인용(별표 원문 대조).

### 미완결(통합 단계 이관)
- validate-data.mjs는 파일 수(62)와 매니페스트 수 일치 및 priority 1..N 연속을 요구한다. invalid-bids 항목 추가로 파일 고유 오류("invalid-bids: manifest 항목이 없습니다")는 해소됐으나, 동일 배치의 다른 2개 파일 미등록과 80·81·82 우선순위 스킴 때문에 전체 검증은 여전히 exit 1이다. 배치 3파일 일괄 등록 + 우선순위 재배열은 통합 담당자 소관.
