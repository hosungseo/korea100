# 기타 계약내용의 변경(과업·공기 조정) — 조사 워크시트 (2026-07-12 기준, 원문 캐시 + 공개 웹 검색)

대상 slug: other-contract-change-adjustment
범위: 물가변동(영 제64조)·설계변경(영 제65조)에 해당하지 않는 "그 밖의 계약내용 변경"(공사기간·운반거리 변경 등)으로 인한 계약금액 조정. 설계변경(design-change-adjustment)·물가변동(price-fluctuation-adjustment)과 중복 배제.

## 확인 못 해 뺀 것 / 조번호 없이 쓴 것

- 정부 입찰·계약 집행기준 제16장(실비의 산정)의 개별 조문 번호: 원문 캐시 파일이 물가변동 관련 장까지만 수록되어 이번 세션에서 조번호를 확정하지 못함 → process legal_basis에는 조번호가 확인된 공사계약일반조건 제23조제1항으로 앵커하고, 집행기준은 canvas legalBasis에만 "제16장 실비의 산정"으로 표기.
- 법률 제19조·시행령 제66조·시행규칙 제74조의3의 항 단위 원문 텍스트: 조문 제명·번호는 캐시 목차 원문으로 확정했으나 항별 전문은 공개 웹 검색으로만 교차 확인 → articleVerification에서 uncheckable로 분류(후속 기계 대조 대상).
