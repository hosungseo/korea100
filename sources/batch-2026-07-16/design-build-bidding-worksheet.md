## 확인 못 해 뺀 것 / 조번호 없이 쓴 것
- 중앙건설기술심의위원회의 근거 조문(건설기술 진흥법 조번호)은 확정 못 함 → legalBasis에는 법률명만, unresolved(title-needs-confirmation) 처리.
- 설계보상비 산정식(예: 설계점수 비례, 1인당 공사예산 상한)은 계약예규/조달청 세부기준 수준의 수치로 시행령 제89조 본문과 별개 → 본문 legal_basis에 수치 미기재, fieldVerification로 이관.
- PQ 신청기간(공고일부터 7일 후 신청, 10일 이상 등)은 일반 사전심사 규정(제13조 계열)이며 turnkey 특정 아님 → 수치 본문 미기재.
- 제85조의2·제87조 세부 결정방법(설계적합최저가·확정가격최상설계·가중치방식)은 조달청/기재부 낙찰자 결정 세부기준 소관 → 노드 action에 개념만, 세부 수치 미기재.
