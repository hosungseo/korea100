## 조사 방식 메모
- 근거 법령 원문 캐시(`sources/law-cache/중소기업제품-구매촉진-및-판로지원에-관한-법률(.md/-시행령.md)`)는 이번 세션에 **목차(조문 번호·제목)만** 담겨 있고 조문 본문은 없었다.
- korean-law MCP `get_law_text`는 이번 세션 내내 `[NOT_FOUND]`를 반환해 조문 본문·항 번호를 직접 대조하지 못했다(환각 방지 프로토콜에 따라 항 번호·본문 문구는 추측하지 않음).
- 따라서 **조문 번호·제목은 캐시 목차(국가법령정보센터 Open API 사본)로 확정**하고, **내용 취지는 WebSearch 교차 확인**만으로 기록한다. 항(제N항) 단위 인용은 확인 불가로 회피하고 조(제N조) 단위로만 인용한다.
- 조문 실존 기계 대조와 항 단위 확정은 통합 시 관리자 재검증(verify:articles)에 맡긴다. verification.status = needs-review.

## 확인 못 해 뺀 것 / 조문 번호 없이 쓴 것
- 판로지원법·시행령의 **항(제N항) 번호**는 조문 본문 미확보로 모두 뺐다. 조(제N조) 단위로만 인용.
- 성능인증 심사절차(적합성심사 평균 70점 이상·7인 이내 심사위원회, 공장심사, 성능검사)는 SMPP/민간 자료 기반 실무 서술이라 조문 근거 없이 절차·fieldVerification로만 기술.
- 성능인증 신청 수수료(약 77만원), 연장신청 접수기간(만료 120~60일 전) 등 세부 수치는 SMPP·민간자료뿐이라 본문에서 제외(fieldVerification 후보).
