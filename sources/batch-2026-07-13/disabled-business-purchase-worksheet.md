## 확인 못 해 뺀 것 / 조문 번호 없이 쓴 것

- 개선 권고·요구의 정확한 항 번호(제9조의2 몇 항): 검색으로 조 수준만 확인 → 조의2 수준으로만 인용, 항 미기재.
- 국가계약법 시행령 제26조제1항제5호의 세부 목(가/나/…): 여러 실무 자료가 "제5호"라고 하나 목 번호는 확정 못 함 → 제5호까지만, fieldVerification 등재.
- 구매계획/구매실적 제출의 법정 기한(연중 몇 월): 확정 증거 없음 → "매년" 수준으로 서술, fieldVerification 등재.
- 확인기관: 지방중소벤처기업청장 발급 / 장애인기업종합지원센터(DEBC) 확인업무 수행이 혼재 → 확인 위탁 구조로 서술, fieldVerification 등재.
