# 직접생산확인 — 조사 워크시트 (2026-07-12, law-cache 원문 + WebSearch 교차)

주 이용자: 조달업체(제조업체). 조사 출발점: 조달청 제조물품 직접생산확인 기준(law-cache 전문 보유).

## 확인 못 해 뺀 것 / 조문 번호 없이 쓴 것

- 조달사업법 제18조제3항·제22조, KNCL 시행규칙 제15조제2항, 판로지원법 제9~11조, 등록규정 제14조·제22조제7항은 캐시 TOC의 제명 또는 다른 원문의 상호인용으로만 확인. 본문 전문 미대조 → 조문 번호는 상호인용 근거가 있는 것만 사용, 본문 미확인 조문(조달사업법 제22조)은 process에서 uncheckable로 계상.
- 부정당업자 제재의 국가계약법 근거 조문 번호는 본문에서 확정 기재하지 않고 "부정당업자 입찰참가자격 제한"을 related로만 연결.
- 직접생산확인 취소 후 재신청 제한 기간(WebSearch에 "3개월" 언급)은 조달청 기준 원문 미확인이라 수치로 쓰지 않음.
