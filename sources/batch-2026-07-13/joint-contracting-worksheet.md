## 확인 못 해 뺀 것 / 조문 번호 없이 쓴 것

- 영 제72조제3항 각 호(제1호·제2호)의 구체적 대상 사업·추정가격 구간: WebSearch가 노이즈(중소기업·소기업 규정과 혼선)로 정확한 각 호 문언 미확정 → 본문은 "각 호의 사업" 수준으로만 서술, fieldVerification/uncheckable 처리.
- 영 제72조제3항 지역업체 관할 단위(특별시·광역시·특별자치시·도·특별자치도) 최신 문언은 WebSearch 스니펫 기준이며 캐시 TOC로만 조문 실존 확인 → 기계 대조 필요.
- 재정경제부/기획재정부 부처 표기: 2026년 개편으로 현행 원문은 재정경제부장관. 기존 파일 관례대로 "기획재정부장관(재정경제부장관)" 병기.
