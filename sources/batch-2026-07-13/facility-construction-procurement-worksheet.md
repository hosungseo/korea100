# 시설공사 조달(수요기관 대행 발주) — 조사 워크시트 (2026-07-12, law-cache + WebSearch)

제도 slug: facility-construction-procurement / type: 발주 / priority: 72
차별화: procurement-request(조달요청·중앙조달)는 요청 여부 판단·계약방법 협의·수수료 정산의 관문을 다룬다.
이 제도는 조달청이 대행 요청받은 **시설공사**를 입찰방식 결정 → 공고 → 자격심사(PQ) → 개찰·적격심사/종심제 → 낙찰 → 공사도급계약 체결까지 집행하고,
착공·공사감독·준공검사·대가지급 등 현장 이행관리는 수요기관이 담당하는 **공사 특화 대행 발주** 절차에 초점을 둔다.

## 확인 못 해 뺀 것 / 조문 번호 없이 쓴 것

- PQ 대상 공사유형(제13조제1항제2호 "기획재정부령이 정하는 공사, 추정가격 200억원 이상")의 구체 공종 목록은 본문 수치로 단정하지 않고 fieldVerification로 돌림.
- 공사계약일반조건의 착공(착공계)·공사감독·준공검사 개별 조문 번호(제17조 등)는 검색 확증 부족 → 조문 번호 없이 "공사계약일반조건"으로만 인용.
- S01(나라장터 전자입찰 접수)은 시스템 단계로 개별 조문 미부여(system 노드, 낮은 confidence).
- 조달청 시설공사 계약업무 처리규정의 개별 조문 번호는 확증 부족 → 조문 번호 없이 규정명 수준 인용.
