## 확인 못 해 뺀 것 / 조문 번호 없이 쓴 것

- 정부 입찰·계약 집행기준의 재공고입찰 관련 **정확한 조·항 번호**는 확인하지 못함 → canvas legalBasis에 조문 없이 규범명으로만 인용, 노드 legal_basis에는 사용하지 않음.
- 제35조 재공고입찰 5일 단축의 정확한 **항·호 위치**(제4항 등)는 검색 스니펫으로만 확인 → 본문은 "재공고입찰은 5일 전까지 단축 공고 가능" 수준으로 서술, articleVerification에서 uncheckable 처리.
- 중소기업제품 구매촉진법상 2인 미만 유찰 특례는 이 제도 범위 밖으로 판단해 제외.
- 경제위기 시 1인 입찰 수의계약 특례(제27조 단서, 기획재정부장관 고시)는 한시·특례라 본문 미반영.
