## 확인 못 해 뺀 것 / 조문 번호 없이 쓴 것 / 낮은 확신

- 이의신청·조정신청 기간(30일): 2026-06-11 시행 개정으로 종전 20일에서 30일로 확대됐다고 다수 보도(정보통신신문, 아주경제, Kim&Chang)하나 원문 항 번호까지 캐시로 대조하지 못함 → 본문·노드에 반영하되 fieldVerification 등재, 노드 confidence 낮춤.
- 이의신청 최소 금액 기준(시행령 제110조): 종합공사 추정가격 10억원(구 30억)·전문공사 1억원(구 3억) 등으로 완화됐다는 검색 결과가 있으나 현행 정확 금액 미확정 → 본문에 구체 금액 확정 기재하지 않고 "시행령 제110조가 정하는 추정가격 기준" 수준으로만 서술, fieldVerification 등재.
- 심사·조정 완료 50일(연장 가능): 정보통신신문·아주경제 보도로 교차 확인되나 원문 대조 미실시 → 노드에 반영·confidence 낮춤.
- 조정안 15일 내 이의 없으면 재판상 화해 효력: 정보통신신문·Kim&Chang 교차 확인 → 반영, confidence 중간.
- 조정 대상 10개 사유(부당특약 포함): Kim&Chang이 열거(국제입찰 범위·부당특약·입찰참가자격·입찰공고·낙찰자 결정·보증금 국고귀속·계약금액 조정·개산계약 정산·지체상금·계약해제) → 배경 서술로만 사용.
