# 예정가격 작성 — 조사 워크시트 (2026-07-12, WebSearch 기반)

대상 slug: estimated-price / 유형: 기준·산정형 / priority 16 / 주 이용자: 계약담당 공무원

법령 원문 캐시(law-cache)에는 국가계약법 3종의 전문 사본이 없고 대상 목록(law-list.json)만 있어, 이번 조사는 전부 WebSearch 교차 대조로 수행했다. 따라서 verification.status는 needs-review로 고정한다.

과제에서 후보로 제시된 조문 배치(시행령 제7조의2 예정가격의 작성 · 제8조 예정가격의 결정방법 · 제9조 예정가격의 결정기준)는 검색으로 대체로 확인되었다. 다만 제7조의2·제7조는 국가/지방 시행령 스니펫이 혼재해 국가 조문 원문 대조까지는 못 했고, 제8조·제9조는 국가법령정보센터 조문정보 링크와 LBOX 국가 시행령 제9조 링크로 조 단위 확인이 됐다. 낙찰선 근거인 법 제10조(경쟁입찰에서의 낙찰자 결정)는 CaseNote·law.go.kr로 확인됐다. 복수예비가격·기초금액·15개·±2%·4개 추첨은 계약예규(예정가격작성기준) 제2장과 나라장터 실무이며 예규 식별자가 없어 unresolved로 돌린다.

## 확인 못 해 뺀 것 / 조문 번호 없이 쓴 것 / uncheckable 처리

- 시행령 제7조의2·제7조: 국가/지방 시행령 스니펫이 섞여 국가 조문 원문 대조를 못 함 → 조 단위로만 인용하고 uncheckableReferences로 계상.
- 시행령 제9조 각 호(거래실례가격=제2호, 원가계산=제3호 등)의 호 번호: 스니펫마다 정렬이 달라 호 단위 미확정 → 조 단위(제9조)로만 인용.
- 시행령 제42조(낙찰자 결정)의 항 번호: 조는 확인되나 항·적격심사 세부는 미확정 → uncheckable.
- 복수예비가격 15개·기초금액 ±2%·4개 추첨 산술평균: 예정가격작성기준(계약예규) 제2장과 나라장터 실무. 예규 식별자 미확보로 unresolved 처리하고, 4개 추첨·투찰 UI는 fieldVerification(운영 확인)로 등재.
- ±2%(국가) vs ±3%(지방) 차이: 본문에는 국가계약 기준 ±2%만 쓰고 지방 수치는 배제.
- 원가계산용역기관 의뢰: 시행령 제9조(원가계산가격) + 예정가격작성기준으로 서술. 국가 시행규칙 세부 조문(용역기관 요건)은 미확정 → 예규/시행령 수준으로만 인용.

## 부처 표기

2026년 정부조직 개편으로 현행 원문은 재정경제부장관/재정경제부령 표기(일반관리비율·이윤율 등 위임). 검색 스니펫은 기획재정부·재정경제부 혼재. 예정가격작성기준 예규도 기획재정부계약예규(최신 제785호, 2025.5.1. 일부개정) 표기가 검색됨. 기존 파일 관례에 따라 "기획재정부장관(재정경제부장관)" 병기.
