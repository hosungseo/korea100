## 확인 못 해 뺀 것 / 조문 번호 없이 쓴 것 / uncheckable 처리

- 시행규칙 제27조: 공사 지명경쟁 지명기준(시공능력·특수기술·특수공법)이라는 취지는 검색으로 확인되나, 정확한 조문 제목과 각 항의 세부 기준·"신용·실적·경영상태" 표현은 원문 대조로 확정하지 못함 → 노드에 article-level "제27조"로만 표기하고 uncheckableReferences로 계상.
- 시행령 제39조(입찰서 제출)·제40조(개찰): 본문 내용(전자 입찰서 제출·개찰·낙찰선언)은 국가계약 절차로 확인되나, 일반경쟁 조사에서 "제39조/제40조"가 지방계약법과 혼재 회수되어 국가 조문 번호 자체를 uncheckableReferences로 처리.
- 시행령 제36조 각 호의 개별 번호(낙찰자 결정방법이 몇 호인지 등)는 미확정 → article-level "제36조"로만 표기.
- 지명통지 방법의 시행규칙 세부 조·서식(통지서 양식)은 재정경제부령 위임 사항으로 확인되나 구체 조문·서식 번호 미확정 → fieldVerification 등재.
- 감사원 통지의 근거 조항: 정부 입찰·계약 집행기준(계약예규)이 "시행령 제23조제2항"을 명시하므로 제23조제2항으로 귀속. 일부 검색 스니펫이 제24조제2항으로 표기한 사례가 있으나 계약예규 명시를 우선함.
- 계약예규(정부 입찰·계약 집행기준): 원문 식별자 미확보 → legalBasis에 행정규칙으로만 등재하고 verification.unresolved 처리.
