## 확인 못 해 뺀 것 / 조문 번호 없이 쓴 것

- 계약이행능력심사 항목 열거가 놓인 시행령 제42조의 정확한 "항" 번호(제5항 등) — 스니펫에서 "제1항에 따른 계약이행능력심사는…"만 확인. 본문·노드는 "제42조"/"제42조제1항" 수준으로만 인용.
- (계약예규) 적격심사기준·조달청 물품구매적격심사세부기준의 조문 번호 — 미확인. law·kind 수준으로만 인용하고 unresolved 처리.
- 시행령 복수예비가격(±3%, 15개 중 다빈도 4개 산술평균) 방식의 근거 조문 — 예정가격 결정 관련 검색에서 방식은 확인되나 정확한 조번호(제8조 등) 미확정 → 본문에서 조번호 없이 "예정가격은 시행령·계약예규가 정하는 방식(복수예비가격 등)으로 결정" 수준으로 서술.
- 국가계약분쟁조정위원회 이의신청 경로는 적격심사 재심사(계약예규)와 별개 절차 → 관련 제도로만 처리, 본 제도 노드에는 예규상 재심사만 반영.

## 수정 이력 (2026-07-12, 검증 심사자 지적 반영)

검증 심사자 지적 5건을 아래와 같이 반영. refuted 항목은 담당자 WebSearch로 재확인 후 수정.

1. **[refuted, must-fix] canvas.applicability — 100억원 경계 근거 층위** : 괄호 서술을 "금액 기준은 계약예규·조달청 세부기준으로 정하며 개정될 수 있다"에서 "공사 100억원 경계는 시행령 제42조가 정하고, 세부 배점·심사 대상은 계약예규·조달청 세부기준으로 정하며 개정될 수 있다"로 수정. 근거: WebSearch 재확인 — easylaw 종합심사낙찰제(csmSeq=519, cnpClsNo=3)·lfind가 "추정가격 100억원 이상 공사는 시행령 제42조제4항에 따른 종합심사낙찰제 대상"으로 명시. 100억원 경계는 대통령령(시행령 제42조제4항)이 직접 정하는 수치이므로 계약예규 소관으로 서술한 상하위 혼동을 바로잡음.

2. **[refuted, must-fix] 물품 종합평점 88점 → 85점** : P07.condition·fieldVerification[1]·verification.notes[0] 세 곳의 물품 예시를 85점으로 일관 수정. 근거: WebSearch 재확인 — 조달청 물품구매적격심사 세부기준상 종합평점 85점 이상 낙찰, 고시금액 미만 낙찰하한율 84.245%(85점과 짝). 88점은 비조달청 기준의 상충 스니펫으로 판단해 삭제. notes에 낙찰하한율 값도 84.245%로 정정.

3. **[unverifiable, should-fix] P06.legal_basis[0] 제42조제1항 문언 귀속** : article을 "제42조제1항"에서 "제42조"로 낮추고, text에 "열거항목이 놓인 정확한 항 번호는 원문 대조 필요" 유보를 명시. 근거: 제42조제1항의 확인된 문언은 "예정가격 이하 최저가 순 계약이행능력·일자리창출 실적 심사·낙찰자 결정"이며, 이행실적·기술능력 등 열거는 워크시트에서도 항 번호 미확정으로 기록. 제1항 귀속 주장을 제거해 워크시트 유보와 일치시킴. (S01·P03·P08·P10의 제42조제1항 인용은 확인된 '최저가 순' 문언이므로 유지.)

4. **[wording, should-fix] canvas.legalBasis[1] 제36조 제목** : "제36조(입찰공고)" → "제36조(입찰공고의 내용)"으로 공식 조문 제목 표기.

5. **[wording, should-fix] 비조문 표기 article — 확인된 것만 조문 번호 부여** : WebSearch로 확인된 항목만 조문 번호 부여. ### articleVerification 집계 재확인
canvas.legalBasis 인용 법령 4건(법률·시행령·계약예규 적격심사기준·조달청 물품구매적격심사 세부기준) → citationEntries=4, explicitCitationEntries=2(법률·시행령만 조문 명시). 조문 참조 4건(법 제10조·영 제42조·영 제36조 = verified 3, 영 제92조의2 = uncheckable 1) → verifiedReferences 3 + missingReferences 0 + uncheckableReferences 1 = articleReferences 4로 합계 일치. 제36조 제목 정정은 verified 상태 불변. sources(법률·시행령 2건) + unresolved(계약예규 적격심사기준·조달청 물품구매적격심사 세부기준 2건)로 canvas.legalBasis 4건 전부 커버, 중복 없음. needs-review 통과 조건(unresolved≥1, uncheckable≥1) 충족. → 집계 수정 불필요.
