## 확인 못 해 뺀 것 / 조문 번호 없이 쓴 것

- 시행령 제21조의 "입찰공고 제한사항·제한기준 명시" 의무가 몇 항인지(제2항/제3항 추정) 검색으로 확정 못함 → 노드에 조문은 article-level "제21조"로만 표기, uncheckable 처리.
- 실적 배수(규모·양/추정가격의 1배 이내)와 30억/3억 기준이 시행규칙 제24조와 제25조 중 어디에 속하는지 스니펫마다 혼재 → 30억/3억은 제24조(대상), 배수·지역·중복금지는 제25조(제한기준)로 귀속하되 배수 항목은 uncheckable 처리하고 본문은 "시행규칙이 정하는 배수 이내"로 완충.
- 지역제한 적용 "고시금액"의 구체 금액(기재부장관 고시)은 개정·연도별 변동이라 본문에 수치 미기재, "고시금액 미만" 수준으로만 서술. (검색에 나온 종합 100억/전문 10억/일반용역 3.3억/건설기술 2.2억 수치는 지방계약법 지역제한이므로 국가계약 본 제도에 사용하지 않음.)
- 물품제조 제한 세부(품질인증 종류 등), 입찰서 제출·개찰의 조문 번호는 검색으로 미확정 → 해당 프로세스 노드에 legal_basis 미부여, system/task로만 서술.
- 판로지원법 조문 번호 미확정 → canvas.legalBasis에는 법령명 수준으로만 등재하고 verification.unresolved 처리.
