## 확인 못 해 뺀 것 / 조문 번호 없이 쓴 것
- 국가계약법 시행령 제26조제1항 내 사회적기업 수의계약의 정확한 호·목(제1호 다목으로 추정)은 시행령 개정으로 재배열 가능 → 본문은 "제26조제1항" 수준으로 인용하고 호·목은 fieldVerification.
- 취약계층 고용비율 30%·1인 견적 5천만원 한도는 우선구매 지침/유권해석 서술이라 본문 수치로 확정하지 않고 유의사항으로만 처리.
- 법정 고정 구매목표 비율(%)은 사회적기업제도에 존재하지 않음(중증장애인 2%·중소기업과 달리 촉진 의무만) → 비율 수치 미기재.
- 시행령 제12조의2 조번호는 원문 캐시 없이 WebSearch만으로 확인 → uncheckable 처리.
