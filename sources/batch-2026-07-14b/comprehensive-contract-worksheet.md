## 확인 못 해 뺀 것 / 조번호 없이 쓴 것

- 국가계약법 시행령 제71조 본문 위임 문구(재정경제부장관/기획재정부장관이 정하는 바)는 WebSearch로 전문 확인 실패. 조번호(제71조 종합계약)는 law-cache 목차로 확인. 본문 취지는 예규 제1조의 명시적 인용으로 역추정 → fieldVerification 등재.
- 조달사업에 관한 법률 시행규칙 제5~10조의 개별 조문 내용은 대조하지 않고, 예규 제9조제3항이 인용한 근거 사슬로만 사용.
- 조달청 종합계약 적격심사 세부기준·심사기간, 공사비 분담방법 세부운영기준(예규 제15조 위임)의 실제 고시·기준 존재 여부·수치는 확인 못 함 → fieldVerification.
