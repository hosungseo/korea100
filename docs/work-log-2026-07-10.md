## 6. 검증 자동화와 안전장치

추가하거나 정비한 주요 스크립트는 다음과 같다.

- `web/scripts/sync-verification.mjs`
  - 100개 제도의 공식 출처 조회 및 JSON 반영
  - 기계 판독 보고서 생성
  - `LAW_OC`가 없거나 CLI 실행 자체가 실패하면 쓰기 전에 중단
- `web/scripts/verify-articles.mjs`
  - 명시 조문 수집, 원문 조회, 결과 집계 및 JSON 반영
  - `LAW_OC`가 없으면 기존 검증 파일을 변경하지 않고 중단
- `web/scripts/lib/article-citations.mjs`
  - 조문 표기 파서와 마스킹 로직
- `web/scripts/article-citations.test.mjs`
  - 조문 파서 회귀 테스트 6건
- `web/scripts/validate-data.mjs`
  - 검증 메타데이터, 공식 URL, 식별자, 집계 일관성 검사

추가한 npm 명령은 다음과 같다.

```bash
npm run validate:data
npm run sync:sources -- --write
npm run verify:articles -- --write
npm run test:article-parser
```
