# Go-To-Market v0

## 전략 문장

먼저 홈페이지 버전으로 `대한민국 제도 100`을 보여준다. 온라인에서 제도별 한 장 샘플을 홍보하고, 독자 반응과 검색 수요를 검증한 다음, 검증된 포맷과 사례를 오프라인 책으로 판다.

## 왜 홈페이지 먼저인가

- 책 제목만으로는 `제도 모델`의 효용이 즉시 전달되기 어렵다.
- 한 장 구조도는 보는 순간 이해되는 상품이라 온라인 샘플과 궁합이 좋다.
- 법령은 계속 바뀌므로 홈페이지가 최신성 보완 채널이 된다.
- 온라인 반응은 출판기획서의 근거가 된다.
- 검색 유입이 가능한 주제다. 예: 환경영향평가 절차, 지방소멸대응기금 구조, 예비타당성조사 흐름.

## 상품 구조

### 1단계: 홈페이지

역할: 시제품, 홍보 채널, 수요 검증, 독자 피드백 수집.

구성:

- 제도 10개 공개 샘플
- 각 제도별 한 장 구조도
- 상태 인식형 업무구조도 예시
- 법령 근거와 기준일
- 뉴스레터/문의/피드백 수집
- 오프라인 책 대기자 등록
- 다음 제도 제작 요청 접수

### 2단계: 온라인 연재

역할: 반복 노출, 신뢰 형성, 검색 자산 축적.

채널:

- 홈페이지 원문
- 브런치 요약
- 링크드인/페이스북 카드형 요약
- 정책 커뮤니티 공유
- 공무원/보좌진/연구자 대상 직접 공유

### 3단계: 오프라인 책

역할: 소장 가능한 레퍼런스, 강의/컨설팅 명함, 출판 상품.

구성:

- 100개 제도 모델
- 분야별 색인
- 제도 유형별 비교
- 법령 구조 읽는 법
- 온라인 부록 QR

## 홈페이지 MVP

### 첫 화면

- 제목: 한 장으로 끝내는 대한민국 제도 100
- 핵심 문장: 기업에는 비즈니스 모델이 있듯이, 국가에는 제도 모델이 있다.
- 바로 보이는 샘플: 환경영향평가 상태 인식형 업무구조도
- CTA: 제도 3개 파일럿 보기, 책 출간 알림 받기

### 필수 페이지

1. 홈
2. 제도 모델이란
3. 환경영향평가
4. 지방소멸대응기금
5. 예비타당성조사
6. 재건축ㆍ재개발 정비사업
7. 첫 10개 제도 목록
8. 제도 100 후보 목록
9. 제도 제작 요청
10. 문의·제도 제작 요청

### 파일럿 제도 페이지 템플릿

- 제도명과 한 줄 요약
- 법령 기준일
- 9칸 제도 모델 캔버스
- 절차 swimlane
- 상태 인식형 업무구조도
- 돈의 흐름
- 문서/데이터 흐름
- 병목과 개혁 포인트
- 관련 제도

## 검증 지표

- 홈페이지 방문자 수
- 제도별 페이지 체류 시간
- 공유/저장/문의 수
- 직접 문의 수
- 다음 제도 제작 요청 수
- 특정 독자군 반응: 공무원, 보좌진, 기자, 연구자, 학생
- 가장 많이 클릭된 제도 후보
- 피드백에서 반복되는 혼란 지점

## 오프라인 책으로 전환하는 기준

- 공개 샘플 10개가 완성된다.
- 20개 제도까지 같은 포맷으로 확장 가능하다는 제작 감각이 생긴다.
- 직접 문의와 제도 제작 요청에서 반복 수요가 확인된다.
- 다음 제도 제작 요청에서 반복 수요가 보인다.
- 특정 독자군에서 `업무에 쓸 수 있다`는 반응이 나온다.
- 출판사에 보여줄 샘플 10장, 목차 100개, 제작 파이프라인, 반응 지표가 준비된다.

## 포지셔닝

이 프로젝트는 행정학 교재가 아니다. 제도 해설 블로그도 아니다. 먼저 온라인에서 작동하는 `제도 모델 미디어`로 시작하고, 그중 검증된 구조를 오프라인 책으로 응축한다.

## 당장 할 일

1. 환경영향평가 웹 샘플 1개를 완성한다.
2. 같은 포맷으로 첫 10개 제도 샘플을 만든다.
3. 홈페이지 MVP를 만든다.
4. 제도 100 후보 목록을 공개 가능한 목차로 다듬는다.
5. 출간 알림/피드백 수집 장치를 붙인다.
6. 다음 제도 제작 요청 접수 장치를 붙인다.
7. 온라인 반응을 기반으로 출판기획서를 보강한다.
